# What is Alar?  

Alar is a full-stack web application created as a pet project. It's an attempt to combine GitHub and Reddit, like "Reddit for programmers."

## Stack

### Frontend

- SvelteKit

### Backend

- FastAPI
- SQLAlchemy
- SQLite
